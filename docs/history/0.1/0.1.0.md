## [v0.1.0] - 2026-03-27

> Note: this entry reflects the repository structure and file names used at the time of `v0.1.0`. Some paths and document names have changed since then.

### 新增功能 (Added)
* 新增游戏内命令面板，支持 `gun <name>`、`passive <name>`、`active <name>`、`item <name>` 文本发放。
* 新增 `Grant` 与 `Random` 按钮，支持不输入文本时直接随机发放一个可用物品。
* 新增结构化日志前缀，区分 `[RandomLoadout][Init]`、`[RandomLoadout][Run]`、`[RandomLoadout][Grant]`、`[RandomLoadout][Command]`。
* 新增 `scripts/extract_randomloadout_log.py`，用于从 `BepInEx` 日志中提取 `RandomLoadout` 自己打印的内容。
* 新增 `docs/development.md`、`docs/deploy.md`、`docs/logging.md`、`docs/commands.md`，按职责拆分项目说明文档。

### 技术改进/重构 (Changed/Refactor)
* 重构 `src/RandomLoadout/` 与 `src/RandomLoadout.Core/` 的职责边界，将选择逻辑、规则模型与运行时接入分层。
* 重构发放配置模型，支持随机规则与按名称指定物品的混搭规则。
* 重构 `scripts/build.ps1` 与 `scripts/test.ps1`，将实际逻辑下沉到 `scripts/build.py` 与 `scripts/test.py`，`ps1` 仅做参数转发。
* 抽取 `scripts/script_common.py` 复用脚本公共逻辑，统一仓库路径、配置参数、`MSBuild` 解析与错误输出风格。
* 调整 `README.md` 为项目总览，并将构建、部署、日志、命令面板等具体说明迁移到 `docs/`。
* 统一 `src/RandomLoadout/Properties/AssemblyInfo.cs` 与 `src/RandomLoadout.Core/Properties/AssemblyInfo.cs` 的程序集版本声明，从四段式 `0.1.0.0` 改为三段式 `0.1.0`。

### 修复 (Fixed)
* 修复进入地牢后首次发放时机，避免在进入地牢前提前给物品导致掉落。
* 修复场景轮询启动期的空引用问题，在 `GameManager` 尚未完全准备好时回退到 Unity 场景名。
* 修复游戏内命令输入框回车不稳定的问题，改为在 `OnGUI` 中直接处理提交事件。
* 修复命令面板与游戏 `Esc` 键冲突的问题，改为使用 `F7` 开关面板。
* 修复命令面板布局与字体可读性问题，将 UI 调整到底部居中并优化字号、间距与暗色主题样式。
