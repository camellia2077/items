## [v0.2.2] - 2026-04-14

### 新增功能 (Added)
* 新增 `src/RandomLoadout/Localization/GuiText.cs` 轻量 GUI 多语言层，支持从 `json5` 语言包读取玩家可见文本，并按游戏语言在 `zh-CN` 与 `en` 之间自动切换。
* 新增默认语言包 `defaults/config/RandomLoadout.localization.en.json5` 与 `defaults/config/RandomLoadout.localization.zh-CN.json5`，集中存储命令面板、物品浏览、角色、货币、Rapid 与调试操作相关文案。

### 技术改进/重构 (Changed/Refactor)
* 调整 `src/RandomLoadout/Plugin.Bootstrap.cs` 的启动流程，在插件初始化时加载 GUI 本地化表，并通过 `tools/tool_common.py` 将语言包纳入默认部署/发布同步清单。
* 调整 `src/RandomLoadout/Commands/InGameCommandController*.cs` 的界面文本来源，统一改为通过本地化 key 渲染按钮、标题、筛选标签、空状态提示与角色页说明文案。
* 调整 `src/RandomLoadout/Commands/GrantCommandExecutionResult.cs` 的消息结构，新增显示消息与日志消息双轨输出，为“玩家可见文本本地化、BepInEx 日志保持英文”提供基础。
* 调整 `src/RandomLoadout.Core/Commands/GrantCommandParser.cs` 与 `src/RandomLoadout.Core/Commands/GrantCommandParseResult.cs`，为输入错误提示补充稳定错误码，供 GUI 侧按语言生成本地化提示。

### 修复 (Fixed)
* 修复命令面板、物品浏览、角色与货币页面大量固定英文硬编码的问题，改为随游戏语言显示中文或英文 GUI 文案。
* 修复 `Grant`、`Random`、`Rapid`、角色切换/解锁与调试按钮执行后只返回英文结果消息的问题，改为优先显示本地化玩家提示。
* 修复新增语言包未自动同步到 `BepInEx/config` 时会导致部署后缺少本地化资源的问题，改为由默认部署链路一并复制。
